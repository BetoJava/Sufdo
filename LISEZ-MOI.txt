Pour y jouer il te faut télécharger python et pygame :

	- Pour python : https://www.python.org/downloads/release/python-392/
	  sur le site tu télécharge le "Windows installer (64-bit)" (dernier normalement)

	- Pour pygame tu fait : "cmd" dans la barre de recherche
	  puis tu met : py -m pip install -U pygame --user
	  puis pour tester si ça a téléchargé : py -m pygame.examples.aliens
	- py -m pip install numpy

	# Les autres librairies suivantes n'ont pas besoin d'êtres téléchargées (ça sera pour les versions suivantes du jeu) #
	- pip install pandas
	- pip install openpyxl xlsxwriter xlrd



Après ça, ça devrait être bon.
Ensuite il te suffit d'ouvrir main.py (dans ce dossier) : ça t'ouvrera la console (utile pour certaines info dans le jeu) et la fenêtre de jeu.
